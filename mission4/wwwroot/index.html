What I Hope to Get Out of IS413

There are several reasons why I want to learn about building web apps in C#. Firstly, C# is a powerful and versatile programming language that is well-suited for web development. It is an object-oriented language with a strong type system, making it easy to write reliable and maintainable code. Additionally, C# is built on top of the .NET framework, which provides a wide range of libraries and tools for web development, such as ASP.NET, which is a framework for building web apps and web APIs.

Secondly, C# is a popular language and has a large community of developers, which means that there are many resources available for learning and troubleshooting. This includes tutorials, forums, and documentation, as well as a wide range of third-party libraries and frameworks. This can make the process of learning to build web apps in C# much easier and more accessible, as you can find help and solutions to problems quickly and easily. Additionally, C# is widely used in the industry, and the demand for developers proficient in C# is high, which can make it a valuable skill for those looking for job opportunities in the field of web development.
